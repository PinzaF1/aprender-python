# Aprender Python

Este repositorio contiene una serie de archivos de ejemplo para aprender los conceptos básicos de Python.

## Lecciones

1. `01_hola_mundo.py` - Hola mundo en Python
2. `02_variables.py` - Variables y tipos de datos
3. `03_bucles.py` - Bucles `for` y `while`
4. `04_condicionales.py` - Condicionales `if` / `else`
5. `05_funciones.py` - Definición y uso de funciones
6. `06_listas.py` - Listas en Python
7. `07_dicionarios.py` - Diccionarios
8. `08_tuplas.py` - Tuplas
9. `09_nuevo_archivo.py` - Ejemplo adicional
10. `10_clases.py` - Clases y objetos
11. `11_excepciones.py` - Manejo de excepciones
12. `12_archivos.py` - Lectura y escritura de archivos
13. `13_herencia.py` - Herencia de clases
