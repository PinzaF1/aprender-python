# Aprender Python 🐍

Este repositorio contiene una colección de scripts y aplicaciones en Python para aprender y practicar.

## Archivos Interactivos

### 31_calculadora_interactiva.py
Una calculadora interactiva que permite realizar las siguientes operaciones:
- ✨ Suma
- ➖ Resta
- ✖️ Multiplicación
- ➗ División
- 🔢 Potencia
- √ Raíz Cuadrada
- 📊 Módulo (Residuo)

**Uso:** `python 31_calculadora_interactiva.py`

### 32_gestor_tareas_interactivo.py
Un gestor de tareas completo con las siguientes funcionalidades:
- ➕ Agregar nuevas tareas
- 📋 Listar todas las tareas
- ✅ Marcar tareas como completadas
- 🗑️ Eliminar tareas
- 📊 Ver estadísticas (total, completadas, pendientes, progreso)
- 🧹 Limpiar tareas completadas

**Uso:** `python 32_gestor_tareas_interactivo.py`

## Contenido General

El repositorio incluye diversos temas de Python como:

- Conceptos básicos (variables, bucles, condicionales)
- Funciones y funciones lambda
- Estructuras de datos (listas, diccionarios, tuplas)
- Programación orientada a objetos (clases, herencia, excepciones)
- Manejo de archivos
- Generadores
- Interfaces gráficas (tkinter)
- Gráficas con matplotlib
- Proyectos interactivos (juego de adivinanza, Minesweeper, Mario Bros)

## Requisitos

- Python 3.6 o superior
- No se requieren dependencias externas para los archivos interactivos

## Instalación

1. Clonar el repositorio:
```bash
git clone <url-del-repositorio>
```

2. Navegar a la carpeta:
```bash
cd aprender-python
```

3. Ejecutar cualquier archivo Python:
```bash
python nombre_archivo.py
```
